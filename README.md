# 🛡️ Java RPG バトルゲーム

## 📖 概要

Javaで作成したコンソール型RPGバトルゲームです。
勇者が複数の敵と戦い、すべて倒すとクリアになります。

---

## 🎮 現在の機能

### 🧍 プレイヤー

* 単体攻撃
* 回復
* 全体攻撃（複数の敵にダメージ）

### 👾 敵

* 複数体出現（スライム・ゴブリン・ドラゴン）
* 生存している敵のみ行動

---

## ⚔️ 戦闘システム

### 🔹 ターン制

1. プレイヤーの行動選択
2. 敵の攻撃
3. 繰り返し

---

### 🔹 コマンド

```
1: 攻撃（単体）
2: 回復
3: 全体攻撃
```

---

### 🔹 ターゲット選択

* 生きている敵のみ表示
* 入力ミス対策あり（例：数字以外・範囲外）

---

## 💡 技術的ポイント

### 🔸 配列とリストの使い分け

* Enemy[] → 敵の管理
* ArrayList → 生存している敵の管理

---

### 🔸 拡張for文（for-each）

```java
for (Enemy enemy : enemies)
```

→ 配列の中身を1つずつ取り出して処理

---

### 🔸 フラグ管理（重要）

```java
private boolean isDead = false;
```

* 倒した瞬間のみ経験値を付与
* 重複取得バグを防止

```java
public boolean checkDead() {
    if (hp <= 0 && !isDead) {
        isDead = true;
        return true;
    }
    return false;
}
```

---

### 🔸 共通クラス設計（オブジェクト指向）

```java
Character（親クラス）
 ├ Player（勇者）
 └ Enemy（敵）
```

* 共通処理をCharacterにまとめる
* Player / Enemyで役割分担

---

### 🔸 全体攻撃（再利用設計）

```java
public void allAttack(Character[] targets)
```

* PlayerでもEnemyでも使用可能
* 拡張性の高い設計
