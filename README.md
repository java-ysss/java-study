* 入力エラー対策（例外処理）

#### できること

1. ゲームを追加（通常 / RPG）
2. 一覧表示
3. クリア済みに変更
4. 削除

### ⑦ 電卓アプリ 🧮

#### 概要

四則演算ができる電卓アプリ
計算履歴の保存・削除機能付き

#### 機能

* 足し算・引き算・掛け算・割り算
* 計算履歴の保存
* 履歴表示
* 履歴の全削除・個別削除
* 最後の履歴削除

#### クラス構成

* Calculator.java → 基本的な電卓処理
* MiniCals.java → 拡張版（履歴機能付き）
* CalcHistory.java → 履歴管理クラス

#### 工夫したポイント

* `ArrayList`で履歴管理
* クラス分割（計算と履歴を分離）
* 入力チェック（`hasNextDouble`）
* 例外処理（`InputMismatchException`）
* 0除算の対策
処理ごとにクラス・メソッドを分割し、保守性を意識して設計

#### できること

1. 計算方法を選択
2. 数値を入力
3. 計算結果を表示
4. 履歴を確認・削除
